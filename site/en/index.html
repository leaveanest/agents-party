Slack-native agent routing OSS

Bring AI agents into the Slack conversations where work already happens.

Agents party is an OSS foundation inspired by Party on Slack. It helps teammates, AI, and operators collaborate in the same Slack workflow across summaries, search, translation, file understanding, and specialist tools.

Product philosophy

AI should not be trapped in a separate app. Agents party turns the Party on Slack philosophy into an operator-owned OSS foundation for Slack workspaces.

Conversation

Teammates, AI, and you

Agents participate in Slack channels and threads, not only private one-to-one chats.

Routing

No single-model assumption

Route across OpenAI, Anthropic, Google, Vertex AI, and other providers by use case and workspace policy.

Operator control

Clear runtime ownership

Slack OAuth, workspace credentials, PostgreSQL, and Terraform boundaries are explicit from the start.
