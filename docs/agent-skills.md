# エージェントスキル一覧

このドキュメントは、`agents-party` に同梱されているビルトインスキルの一覧です。
スキル文書の実体は `skills/*/SKILL.md` です。アプリケーション runtime への組み込みは TypeScript 側の `src/agents/` または `src/providers/` で実装します。

## 現在ロード対象のスキル

### 運行・オペレーション

### `dispatch-triage`

- 概要: 断片的な Slack 投稿やハドル要約から、遅延・未着・集荷漏れ・再配車のような運行トラブルを初動整理するスキルです。
- 主な用途: 事実の切り出し、緊急度判断、直近の確認項目、次アクションの提示。

### `handover-brief-builder`

- 概要: Slack スレッドやハドルメモから、引き継ぎ向けの短い運用サマリを作るスキルです。
- 主な用途: シフト交代、ルート引き継ぎ、チーム移行時の current state / open items / waiting on の整理。

### `shipper-communication-drafter`

- 概要: 粗い内部メモや状況共有から、荷主向けの簡潔な連絡文面を下書きするスキルです。
- 主な用途: 現状、対応中の内容、次回更新見込みを外部向けに落ち着いた文面へ整えること。

### 旅行・滞在・移動

### `airport-transfer-planner`

- 概要: 空港とホテル間の移動を、実行しやすい短い移動提案にまとめるスキルです。
- 主な用途: 到着時刻、荷物量、深夜到着、乗り換え負荷を踏まえた現実的な移動手段の比較。

### `area-safety-and-convenience-checker`

- 概要: エリアや駅周辺が滞在先や待ち合わせ場所として実用的かを評価するスキルです。
- 主な用途: 深夜の利便性、駅アクセス、歩きやすさ、明確な注意点の整理。

### `budget-stay-optimizer`

- 概要: 宿泊費だけでなく、移動コストや不便さまで含めて実質コストで比較するスキルです。
- 主な用途: 安さ重視の宿選び、費用対効果比較、タクシー代や移動負荷込みの意思決定。

### `family-stay-advisor`

- 概要: 家族連れや移動負荷に敏感な旅行者向けに、滞在先の実用性を評価するスキルです。
- 主な用途: ベビーカーや荷物の負荷、部屋構成、駅導線、深夜到着のしやすさの比較。

### `itinerary-gap-checker`

- 概要: 宿泊、会食、会議、移動を含む旅程の現実的な抜け漏れや無理を洗い出すスキルです。
- 主な用途: 無理な移動、深夜リスク、バッファ不足、宿の立地ミスの早期発見。

### `lodging-search-advisor`

- 概要: 宿泊候補や滞在エリアを、立地と実用性を重視して短い推奨にまとめるスキルです。
- 主な用途: どこに泊まるべきか、駅や目的地へのアクセス、予算や深夜到着のトレードオフ整理。

### `meeting-location-advisor`

- 概要: 参加者のアクセス負荷を踏まえて、実用的な待ち合わせ場所やエリアを提案するスキルです。
- 主な用途: 中間地点の選定、集まりやすさ、乗り換え負荷、天候や終電リスクの考慮。

### 調査

### `web-research-analyst`

- 概要: 最新情報や公式情報を優先して、出典付きの短い調査回答にまとめるスキルです。
- 主な用途: 最新事実の確認、公式ドキュメント確認、比較調査、リンク付き要約。

## 実装上の補足

- `skills/` 配下の `SKILL.md` が実体です。
- Python runtime は削除済みです。新しい skill 連携を追加する場合は TypeScript runtime に明示的な読み込み・実行経路を追加します。
- `skills/` の文書は runtime から自動的に読み込まれるとは限りません。対象 specialist に組み込むまで参照資産として扱います。
