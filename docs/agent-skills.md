# エージェントスキル一覧

このドキュメントは、`agents-party` に同梱されているビルトインのエージェントスキル一覧です。  
スキル名の正本は `src/agents_party/agents/skills/catalog.py` です。

## ビルトインスキル

### `dispatch-triage`

- 概要: Slack の投稿やハドルの断片的な報告から、配車・運行トラブルの初動整理を行うスキルです。
- 想定用途: 遅延、ドライバー未着、集荷漏れ、ルート障害、至急の再配車といった状況で、情報が不完全なままでも整理を始めたい時に使います。
- 主な出力: 状況要約、緊急度、想定影響、すぐ確認すべき事項、次のアクション。

### `handover-brief-builder`

- 概要: Slack スレッドやハドルメモから、引き継ぎ用の短い運用サマリを作るスキルです。
- 想定用途: シフト交代、ルート引き継ぎ、担当者交代のタイミングで、現状と未解決事項を短く整理したい時に使います。
- 主な出力: 現在の状況、未解決事項、回答待ち、注意点、次のアクション。

### `shipper-communication-drafter`

- 概要: 社内向けの粗いメモや状況共有から、荷主向けの連絡文面を作るスキルです。
- 想定用途: 社内の運行情報やハドル要約をもとに、外部向けの簡潔なステータス連絡へ整えたい時に使います。
- 主な出力: そのまま送れるメッセージ案、必要に応じた内部メモ。
